#### Font sizes

    |\tiny         |5     |6     |6     |
    |\scriptsize   |7     |8     |8     |
    |\footnotesize |8     |9     |10    |
    |\small        |9     |10    |10.95 |
    |\normalsize   |10    |10.95 |12    |

    |\Large        |14.4  |14.4  |17.28 |
    |\LARGE        |17.28 |17.28 |20.74 |
    |\huge         |20.74 |20.74 |24.88 |
